Comment obtenir ces informations ?
1. Va sur supabase.com → ton projet
2. Menu gauche → Settings → API
3. Copie Project URL et anon public key

Créer la table (SQL Editor) :
Colle et exécute le SQL ci-dessous dans SQL Editor de Supabase :
create table prospects (
  id bigserial primary key,
  nom text not null,
  ville text,
  tel text,
  email text,
  contact text,
  volume text,
  tier text default '1',
  statut text default 'a-contacter',
  notes text,
  next_contact date,
  kits integer default 15,
  created_at timestamptz default now()
);
alter table prospects enable row level security;
create policy "public_all" on prospects for all using (true) with check (true);
